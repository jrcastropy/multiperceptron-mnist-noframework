# Multiperceptron Nueral Network without Framework

## Description

A simple Multiperceptron Nueral Network using only numpy and pandas for the MNIST Dataset

## Getting Started

### Dependencies

* Python >= 3.6+
* Windows >= 10

### Installing

```
pip install -r requirements.txt
```

### Executing program

* run the jupyter notebook
